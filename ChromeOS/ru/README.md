# Sunny-branded russian phonetic keyboard layout
 Sunny-branded russian phonetic keyboard layout and input method for Chrome OS.

Based on https://github.com/google/extra-keyboards-for-chrome-os/tree/master

## Installation

### GitHub

1. Open a new tab and go to chrome://extensions.
2. Toggle "Developer mode" on (it is in the top right corner of the extensions
   page).
3. Click "Load unpacked extensions...".
4. Pick the directory containing the manifest for the extension you want to
   enable and click "Open".
5 Go to "Settings/Device/Keyboard and inputs/Input settings" and add input method to the input methods.
